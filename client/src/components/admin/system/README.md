# Admin System

| Component   | System             |
| :---------  | :---------------   |
| SchoolYear  | จัดการปีการศึกษา      |
| Upload      | อัปโหลดข้อมูลนักศึกษา  |
| Statistic   | สถิติการลงทะเบียน      |
